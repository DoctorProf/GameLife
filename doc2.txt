## Что говорит код муравьёв (ключевой момент)

Этот проект — **переломный момент в твоём стиле**:

```cpp
int main()
{
	InitWindow(1280, 720, "Ants");
	SetTargetFPS(60);

	Simulation simulation = Simulation(WIDTH, HEIGHT, {5,5}, 1000, 0.995f, 0.1f, 120.f, 30, 1, 20, 1000, 500.f);
	simulation.init();

	while (!WindowShouldClose())
	{
		float dt = GetFrameTime();
		simulation.update(dt);
		// ... отрисовка
	}
}
```

**Что здесь гениально:**

1. **main() чистый** — 30 строк вместо 500. Вся логика в `Simulation`.
2. **Параметризация** — все параметры муравьёв передаются в конструктор. Можно экспериментировать.
3. **dt (delta time)** — правильный игровой цикл, независимый от FPS.
4. **Декомпозиция** — `simulation.update(dt)` и вся сложность внутри.
5. **Отрисовка отдельно** — только рисование в main, логика скрыта.

Это уже **код человека, который понимает архитектуру**.
